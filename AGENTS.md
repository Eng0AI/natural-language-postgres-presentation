# Natural Language Postgres Presentation

Natural language to SQL with a presentation-focused UI for showcasing data insights with PPT-style visualizations powered by Reveal.js.

## Tech Stack

- Next.js 15
- React 19
- Vercel AI SDK
- OpenAI
- PostgreSQL
- Reveal.js
- TanStack Query
- Tailwind CSS
- Recharts
- Framer Motion
